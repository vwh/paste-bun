# Paste Bun

Pastebin alternative built with Elysia, powered by the Bun runtime and SQLite for fast and lightweight text and code snippet sharing.

</div>

## 🚀 Development

To install dependencies, run:

```
bun install
```

To start the development server, run:

```
bun run dev
```
